Introdução - Básico do Git

Olá bem vindos a um documento simples para o uso do Git, aqui iremos ver
os pricipais comandos para você trabalhar com a ferramenta de versionamento
de código da Microsoft.

Primeiro vamos ver os comando básicos para commitar o seu código no gitHub

  1. Git config --global user.name "god git"
  2. Git config --global user.email johndoe@example.comAutenticação no git
  3. Git clone -> clonar o repositorio para uma pasta local
  4. Git checkout -b -> Cria uma nova branch (para trabalhar com versões)
  5. Git status -> Saber o que foi alterado no seu documento
  6. Git add -A -> Adiciona todas as auterações de uma só vez
  7. Git commit -m -> Commit os documentos no repositorio (bom certificar que esteja tudo certo)
  8. Git push origin (nome da branch) -> Ele manda as suas modificações para a conta cadastrada no gitHub
